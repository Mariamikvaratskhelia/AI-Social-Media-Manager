# AI-Social-Media-Manager
Use AI tools to create content for businesses.

What it does:
1) Instagram Post + Caption - Generates 3 A/B/C caption variants + Canva visual tip
2) Reels Script - Full script with timestamps + CapCup editing notes
3) Content Calendar - Multi-week calendar with pillars, hooks, and visual notes
4) Hashtag Strategy - 30 hashtags by category (Mega/Large/Medium/Niche) + branded ideas
5) Full Strategy Report - 90-day roadmap with ChatGPT, Canva & CapCut workflow

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Set your OpenAI API key in `.env` file: `OPENAI_API_KEY=your_key_here`
3. Run the app: `streamlit run app.py`
